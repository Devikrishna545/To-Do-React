## ğŸ“± How to Use

1. **Adding a Task**: Type your task in the input field and click the plus (+) icon
2. **Completing a Task**: Check the checkbox next to a task to mark it as completed
3. **Deleting a Task**: Click the trash icon to remove a task
4. **Viewing Tasks**: 
   - Main area shows all tasks with their status
   - Right panel displays completed tasks
   - Left panel shows pending tasks
